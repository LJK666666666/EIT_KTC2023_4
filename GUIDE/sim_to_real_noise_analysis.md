# 电阻抗断层成像训练数据生成中的仿真-真实差距分析与结构化噪声注入

## 1 引言

基于深度学习的电阻抗断层成像（EIT）图像重建依赖于大规模仿真训练数据，因为带标注的真实测量数据极为稀缺。然而，用于数据生成的二维有限元正向模型与真实的三维物理测量过程之间存在根本性的**仿真-真实差距（Sim-to-Real Gap）**。该差距表现为仿真电压测量值与真实电压测量值之间的系统性偏差，导致模型在真实数据上的性能下降。

本文描述了一种方法：（1）利用少量带标注的真实测量数据量化仿真-真实差距；（2）提取系统误差的结构特征；（3）将其注入训练数据生成流程。该方法遵循**数字孪生残差分析（Digital Twin Residual Analysis）**框架，与 EIT 文献中的近似误差方法（Approximation Error Method, AEM）密切相关。

## 2 问题建模

### 2.1 正向模型

EIT 正问题通过完备电极模型（Complete Electrode Model, CEM）从电导率分布 $\sigma(x,y)$ 计算边界电压 $U \in \mathbb{R}^M$：

$$U_{\text{sim}} = \mathcal{F}(\sigma, z)$$

其中 $\mathcal{F}$ 表示有限元正向求解器，$z$ 为接触阻抗向量。在 KTC2023 实验配置中，32 个电极和 76 种电流注入模式共产生 $M = 2356$ 个测量通道。

### 2.2 仿真-真实差距

真实测量值 $U_{\text{real}}$ 与仿真预测值 $U_{\text{sim}}$ 之间的差异来源于多种误差：

$$U_{\text{real}} = U_{\text{sim}} + \underbrace{E_{\text{sys}}}_{\text{系统误差}} + \underbrace{E_{\text{rand}}}_{\text{随机噪声}}$$

其中：
- $E_{\text{sys}}$：固定系统误差（二维/三维模型截断误差、电极位置偏差、网格离散化误差、未建模的边界效应）。该分量在同一物理系统上的**所有测量中保持不变**。
- $E_{\text{rand}}$：随机测量噪声（热噪声、ADC 量化噪声、接触阻抗波动）。该分量随测量变化。

此前的仿真流程仅将 $E_{\text{rand}}$ 建模为具有信号依赖方差的独立高斯噪声：

$$\text{Var}[E_{\text{rand},i}] = \left(\frac{\eta_1}{100} |U_{\text{ref},i}|\right)^2 + \left(\frac{\eta_2}{100} \max_j |U_{\text{ref},j}|\right)^2$$

其中 $\eta_1 = 0.05\%$，$\eta_2 = 0.01\%$。系统分量 $E_{\text{sys}}$ 被完全忽略。

## 3 数字孪生残差分析

### 3.1 电导率标定

KTC2023 数据集提供了 4 条训练样本和 21 条评估样本，均附有已知的三类分割掩码（真值）。每个真值掩码定义三个空间区域：

$$\sigma(x,y) = \sigma_{\text{bg}} \cdot M_0 + \sigma_1 \cdot M_1 + \sigma_2 \cdot M_2$$

其中 $M_k$ 为第 $k$ 类的二值掩码，$\sigma_{\text{bg}}, \sigma_1, \sigma_2$ 为未知的标量电导率。该参数化将逆问题从高维场简化为**每个样本至多 3 个标量未知数**。

背景电导率 $\sigma_{\text{bg}}$ 首先通过参考测量 $U_{\text{ref}}$（均匀介质，无包含体）估计：

$$\hat{\sigma}_{\text{bg}} = \arg\min_{\sigma} \| \mathcal{F}(\sigma \cdot \mathbf{1}, z) - U_{\text{ref}} \|^2$$

得到 $\hat{\sigma}_{\text{bg}} = 0.804$ S/m。在 `--fix-bg` 选项下，该值在后续所有样本拟合中保持固定，仅优化各样本的 $\sigma_1, \sigma_2$：

$$(\hat{\sigma}_1, \hat{\sigma}_2) = \arg\min_{\sigma_1, \sigma_2} \| \mathcal{F}(\sigma_{\text{bg}} M_0 + \sigma_1 M_1 + \sigma_2 M_2, z) - U_{\text{real}} \|^2$$

优化采用 L-BFGS-B 算法，设置硬边界约束（$\sigma_1 \in [10^{-4}, 5]$，$\sigma_2 \in [0.1, 50]$）。单次正向求解约需 60 ms，收敛通常需要 50-150 次函数评估（每个样本约 3-15 秒）。

### 3.2 残差提取

标定完成后，计算最优拟合仿真电压 $U_{\text{sim}}^*$，并提取残差：

$$E_{\text{gap}} = U_{\text{real}} - U_{\text{sim}}^*$$

该残差包含参数化电导率模型**无法解释的所有误差来源**：模型截断误差、电极几何失配以及随机噪声。

### 3.3 标定结果

表 1 汇总了 25 个样本（4 条训练 + 21 条评估）的拟合电导率值。

| 参数 | 范围 | 均值 | 单位 |
|------|------|------|------|
| $\sigma_{\text{bg}}$（背景） | 0.791 -- 0.868 | 0.804 | S/m |
| $\sigma_1$（低导电包含体） | $\approx 0$（触及下界） | 0.001 | S/m |
| $\sigma_2$（高导电包含体） | 0.3 -- 10.3 | 5.0 | S/m |

低导电包含体的电导率趋近于零（近绝缘），与物理实验设置一致。高导电包含体跨越较大范围（3.5--10.3 S/m），大部分集中在 4--6 S/m。

## 4 噪声分布分析

### 4.1 全局统计

表 2 汇总了残差特征。

| 指标 | 训练集 (4) | 评估集 (21) | 合计 (25) |
|------|-----------|------------|----------|
| 噪声标准差 | 0.0355 | 0.0388 | 0.0383 |
| 信噪比范围 | 21.3 -- 22.6 dB | 19.3 -- 23.4 dB | 19.3 -- 23.4 dB |
| 样本间相关系数 | 0.965 | 0.926 | 0.933 |
| 相对拟合误差 | 7.7 -- 8.5% | 7.4 -- 10.3% | 7.4 -- 10.3% |

### 4.2 关键发现

**（1）残差以系统误差为主，而非随机噪声。**

所有 25 个样本的 $E_{\text{gap}}$ 之间的 Pearson 相关系数为 **0.933**（范围：0.813--0.992）。这意味着约 93% 的残差能量在样本间共享，代表一个**不依赖于成像体模的固定空间偏差模式**。该偏差主要源于二维/三维模型截断：正向求解器假设为二维截面，而实际容器为三维结构。

**（2）残差不服从高斯分布。**

Kolmogorov-Smirnov 检验拒绝正态性假设（$p = 2.5 \times 10^{-121}$）。分布具有 **15.8 的超额峰度**（高斯分布为 0），表明存在重尾和更尖锐的中心峰。这与集中型系统分量和稀疏异常通道的混合特征一致。

**（3）噪声具有信号依赖性。**

按 $|U_{\text{ref}}|$ 对通道分箱分析显示，信号幅值与噪声标准差之间存在明显的正相关：大信号通道（$|U| > 0.9$）的噪声标准差约为 0.06，而小信号通道（$|U| < 0.02$）的噪声标准差约为 0.02。这验证了信号依赖噪声模型的结构合理性，但实际噪声水平比先前仿真参数所暗示的约大 30 倍。

**（4）特定通道表现出异常噪声。**

索引 1200 和 1900 附近的通道持续表现出较高噪声（标准差高达 0.08），提示某些电极对存在硬件特定的伪影。

## 5 结构化噪声注入

### 5.1 分解

基于上述分析，将仿真-真实差距分解为：

$$E_{\text{gap}} = \underbrace{\bar{E}}_{\text{系统偏差}} + \underbrace{E_{\text{gap}} - \bar{E}}_{\text{残余随机分量}}$$

其中 $\bar{E} = \frac{1}{N}\sum_{i=1}^{N} E_{\text{gap}}^{(i)} \in \mathbb{R}^{2356}$ 为从全部 $N=25$ 个标定样本计算的均值偏差向量。该向量保存为 `data/systematic_bias.npy`。

### 5.2 改进的数据生成流程

训练数据生成流程（`scripts/generate_data.py`）进行如下增强：

$$U_{\text{train}} = \mathcal{F}(\sigma, z) + \underbrace{\bar{E}}_{\text{系统偏差}} + \underbrace{L_n \cdot \epsilon}_{\text{随机噪声}}$$

其中 $L_n = \text{Cholesky}(\Gamma_n)$ 为现有信号依赖噪声协方差因子，$\epsilon \sim \mathcal{N}(0, I)$。

系统偏差 $\bar{E}$ 在 `data/systematic_bias.npy` 文件存在时自动加载，也可通过 `--sys-bias <path>` 手动指定。传入 `--sys-bias none` 可禁用该功能。

### 5.3 更新的物理参数

基于标定结果，以下仿真参数进行了更新：

| 参数 | 原值 | 更新值 | 依据 |
|------|------|--------|------|
| $\sigma_{\text{bg}}$ | 0.745 S/m | **0.804 S/m** | 由 $U_{\text{ref}}$ 拟合得到 |
| $\sigma_2$（高导电） | $U[5.0, 6.0]$ | **$U[4.0, 6.0]$** | 覆盖大部分标定值 |
| 系统偏差 | 无 | **$\bar{E}$（2356 维向量）** | 25 个标定残差的均值 |

低导电电导率范围（$U[0.025, 0.125]$ S/m）保持不变，标定值与近零电导率一致。

## 6 局限性

1. **标定样本量小。** 仅有 25 个样本可用于估计 $\bar{E}$。系统偏差估计是稳健的（样本间相关性高），但随机分量的协方差结构估计不充分。从 25 个样本中无法可靠估计完整的 $2356 \times 2356$ 协方差矩阵。

2. **标量电导率假设。** 假设每个包含体区域具有均匀电导率。实际上，包含体内部的电导率可能存在空间变化，且像素级掩码边界可能与物理边界不完全对齐。

3. **固定接触阻抗。** 标定过程中接触阻抗 $z = 10^{-6}$ $\Omega$ 保持固定。该假设值与真实接触阻抗之间的任何失配都被吸收到 $E_{\text{gap}}$ 中。

4. **二维正向模型。** 主要系统误差来源是二维/三维模型失配，该失配产生随体模几何变化的结构性偏差。对所有样本使用单一均值 $\bar{E}$ 是一种近似；原则上，偏差取决于电导率分布。然而，较高的样本间相关系数（0.93）表明该近似是合理的。

## 7 使用方法

```bash
# 步骤 1：标定真实数据并提取系统偏差
python scripts/calibrate_real_data.py --fix-bg

# 步骤 2：计算并保存均值偏差向量
python -c "
import numpy as np
E = np.load('results/calibration_X/E_gap_all.npy')
np.save('data/systematic_bias.npy', E.mean(axis=0))
"

# 步骤 3：生成训练数据（系统偏差自动加载）
python scripts/generate_data.py --level 1 --num-images 10000 --save-measurements --workers 8
```
